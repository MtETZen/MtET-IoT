# 🌸 Anime Style Calculator

เครื่องคิดเลขสไตล์อนิเมะที่มาพร้อมกับดีไซน์โปร่งแสง (Glassmorphism) และภาพพื้นหลังน่ารักๆ โปรเจกต์นี้มีทั้งเวอร์ชัน Web (HTML/CSS/JS) และ Desktop Application (Python/PyQt5)

## ✨ ฟีเจอร์หลัก (Features)
* 🎨 **Glassmorphism UI:** ดีไซน์หน้าต่างเครื่องคิดเลขแบบกระจกใสเบลอ สวยงามทันสมัย
* 🧮 **Basic Calculation:** รองรับการบวก ลบ คูณ หาร และหาเปอร์เซ็นต์พื้นฐานครบถ้วน
* 🔙 **Clear & Delete:** ปุ่ม AC (เคลียร์การคำนวณทั้งหมด) และ DEL (ลบตัวเลขทีละตัว)
* 🖼️ **Custom Background:** สามารถเปลี่ยนรูปภาพพื้นหลังสไตล์อนิเมะได้ตามใจชอบผ่าน URL

## 🛠️ เทคโนโลยีที่ใช้ (Technologies)
* **เวอร์ชันเว็บ:** HTML5, CSS3, Vanilla JavaScript
* **เวอร์ชันเดสก์ท็อป:** Python 3, PyQt5 (สำหรับสร้าง GUI)

---

## 🚀 วิธีการติดตั้งและใช้งาน (How to Run)

### 🌐 สำหรับเวอร์ชัน Web (HTML)
1. ดาวน์โหลดหรือสร้างไฟล์ `index.html` แล้วนำโค้ดไปวาง
2. ดับเบิลคลิกเปิดไฟล์ด้วยเว็บเบราว์เซอร์ (Chrome, Edge, Safari ฯลฯ) เพื่อใช้งานได้ทันที

### 💻 สำหรับเวอร์ชัน Desktop (Python)
1. ตรวจสอบให้แน่ใจว่าได้ติดตั้ง Python ลงในเครื่องแล้ว
2. เปิด Terminal หรือ Command Prompt และติดตั้งไลบรารี PyQt5 ด้วยคำสั่ง:
   ```bash
   pip install PyQt5
# 🏃‍♂️ คู่มือการเล่นเกมวิ่งหนี (Runaway Dodge Game)

มินิเกมวิ่งหลบสิ่งกีดขวางที่เขียนด้วย Python และ Pygame ยิ่งเอาชีวิตรอดได้นาน เกมจะยิ่งเร็วและท้าทายมากขึ้น!

## 🎮 วิธีการเล่น (How to Play)
* **เป้าหมาย:** บังคับกล่องสีน้ำเงินหลบกล่องสีแดงที่ตกลงมาให้ได้นานที่สุด
* **การควบคุม:** 
  * กดปุ่ม `<-` (ลูกศรซ้าย) เพื่อวิ่งไปทางซ้าย
  * กดปุ่ม `->` (ลูกศรขวา) เพื่อวิ่งไปทางขวา
* **เริ่มเกมใหม่:** เมื่อแพ้ สามารถกดปุ่ม `Spacebar` เพื่อเริ่มเล่นรอบใหม่ได้ทันที
